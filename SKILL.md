# 基金涨跌预估技能（Agent 执行手册）

## 触发场景

- 用户询问持仓基金的实时涨跌 / 估算盈亏
- 日报附加入当日涨跌预估模块

**不要用于**：选基推荐、基金比较、建仓时机判断、加仓/减仓建议、买入批次跟踪、择时决策。**本 Skill 只处理 memory 中的已有持仓基金**。

---

## 数据隐私原则

**持仓数据（份额/成本）从 memory（用户 profile）读取**，禁止硬编码写入技能文件。

---

## 执行步骤

### Step 1 — 判断是否为交易时段

- **交易时段**：周一至周五 09:30–11:30、13:00–15:00
- 非交易时段：返回提示"当前非交易时段，估算净值已停止更新，最终以基金公司披露净值为准"

### Step 2 — 获取数据（并行）

**数据源路由规则**：

| 基金类型 | 数据源 | 说明 |
|---------|--------|------|
| 场外基金（含主动管理型） | 天天基金 JSONP | `fundgz.1234567.com.cn`，最可靠 |
| 场内 ETF 联接基金 | 同花顺 HQ 页面 | `hq.hexun.com`，降级至天天基金 |

> ⚠️ **降级规则**：同花顺 HQ 页面数据为空 → 直接降级至天天基金 JSONP，不要反复重试。

**天天基金 JSONP 接口**：
```bash
curl -s "https://fundgz.1234567.com.cn/js/{基金代码}.js?rt=$(date +%s)000" \
  -H "User-Agent: Mozilla/5.0" \
  -H "Referer: https://fund.eastmoney.com/"
```
返回解析：`gsz`(估算净值) | `dwjz`(昨日净值) | `gszzl`(估算涨跌幅%)

### Step 3 — 计算结果

#### 预估计算口径（固定，必须严格遵守）

```
单基金日涨跌率  = (gsz − dwjz) / dwjz
单基金日盈亏    = (gsz − dwjz) × 持有份额
组合日盈亏      = Σ((gsz − dwjz) × 持有份额)
组合日涨跌率    = Σ((gsz − dwjz) × 持有份额) / Σ(dwjz × 持有份额)
```

> ⚠️ **绝对禁止**：用成本价 `cost` 作为日涨跌率或日盈亏的分母。日涨跌率的分母必须是 `dwjz`（昨日净值）。成本价 `cost` 只可用于单独标注的持仓**累计收益**，且不得混入当日估算。
>
> ⚠️ **ETF / 指数 / 板块行情不得替代场外基金估算净值参与盈亏计算**。

#### 持仓读取结果

- 基金代码、基金名称、持有份额、缺失字段
- memory 缺少份额时：只输出涨跌率，注明"份额缺失，无法计算盈亏"

#### 单基金当日预估输出

- 昨日净值（dwjz）
- 当前估算净值（gsz）
- 日涨跌率
- 日盈亏金额（份额 × (gsz − dwjz)）
- 估算时间
- 数据源

#### 组合汇总输出

- 总估算盈亏
- 组合估算涨跌率
- 可计算 / 不可计算基金数量
- 缺失或降级说明

### Step 4 — 格式化输出

```markdown
## 📊 今日行情速览（YYYY-MM-DD HH:MM）

> 数据源：天天基金 JSONP
> ⚠️ 非交易时段提示：估算值非实时，最终以基金公司披露净值为准

### 📉 持仓基金涨跌

| 基金 | 昨日净值 | 估算净值 | 日涨跌 | 日盈亏 | 估算时间 |
|------|---------|---------|--------|--------|---------|
| 万家有色C | 1.8191 | 1.9023 | +4.57% | +914.55 | 14:32 |
| ... | ... | ... | ... | ... | ... |

**组合总估算盈亏**：{总估算盈亏} 元（{组合日涨跌率}）
**可计算/不可计算**：{N} / {M} 只

### ⚠️ 风险提示
- 盘中估算净值非最终净值，可能与基金公司披露存在偏差
- 本输出不构成任何买卖建议
- 最终收益以基金公司披露的当日净值为准
```

## 异常边界处理

| 情况 | 处理方式 |
|------|---------|
| memory 缺少份额 | 只输出涨跌率，注明"份额缺失，无法计算盈亏" |
| 缺少估算净值（gsz=null/0） | 输出"不可计算"，标注低可信度 |
| 估算时间过旧（>2h） | 输出"估算时间过旧，可能不反映当前市场" |
| 数据源降级（接口超时） | 标注"数据源降级，可信度低"，跳过该基金 |
| 非交易时段 | 提示"估算净值已停止更新，非最终净值" |
