# 基金实时涨跌预估技能（Agent 执行手册）

## 触发场景

- 用户询问基金持仓的实时涨跌 / 盈亏
- 每日定时日报推送

**不要用于**：选基推荐、基金比较、长期业绩分析、投资决策建议。

---

## 数据隐私原则

**持仓数据（份额/成本）从 memory（用户 profile）读取**，禁止硬编码写入技能文件。
路由映射表（基金→跟踪标的 ETF）只含代码映射，无份额/成本，属于公开信息。

## 执行步骤

### Step 1 — 判断是否为交易时段

- **交易时段**：周一至周五 09:30-11:30、13:00-15:00
- 非交易时段：返回提示"当前非交易时段，估算净值已停止更新"

### Step 2 — 获取数据（并行）

**数据源路由规则**：

| 基金类型 | 数据源 | 说明 |
|---------|--------|------|
| 场外基金（含主动管理型） | 天天基金 JSONP | `fundgz.1234567.com.cn`，最可靠 |
| 场内 ETF 联接基金 | 同花顺 HQ 页面 | `hq.hexun.com`，降级至天天基金 |

> ⚠️ **降级规则**：同花顺 HQ 页面数据为空 → 直接降级至天天基金 JSONP，不要反复重试。

**天天基金 JSONP 接口**：
```bash
curl -s "https://fundgz.1234567.com.cn/js/{基金代码}.js?rt=$(date +%s)000" \
  -H "User-Agent: Mozilla/5.0" \
  -H "Referer: https://fund.eastmoney.com/"
```
返回解析：`gsz`(估算净值) | `dwjz`(昨日净值) | `gszzl`(估算涨跌幅%)

### Step 3 — 计算结果

```python
# 计算公式（严格遵守）
今日市值 = gsz × 持有份额
昨日市值 = dwjz × 持有份额
成本市值 = cost × 持有份额

日涨跌%  = (gsz / dwjz − 1) × 100   # 分母是 dwjz，不是 cost
日盈亏    = (gsz − dwjz) × 持有份额
总涨跌%  = (gsz / cost − 1) × 100    # 分母是 cost
总盈亏    = (gsz − cost) × 持有份额

# 验证
Σ(gsz × 持有份额) == 总资产（误差<1元）
```

> ⚠️ **PITFALL**：绝对不能用 `gsz / cost − 1` 算日涨跌；场内 ETF 实时价格≠场外基金的 dwjz。

### Step 4 — 格式化输出

```markdown
## 📊 今日行情速览（YYYY-MM-DD HH:MM）

### 📉 持仓基金涨跌

| 基金 | 日涨跌 | 日盈亏 | 总涨跌 | 总盈亏 |

| 万家有色C | +4.46% | +858.67 | +0.61% | +122.04 |
| ... | ... | ... | ... | ... |

**总资产**：{总资产} 元（成本 {总成本} 元）
**今日盈亏**：{日总盈亏} 元（{日总涨跌%}）
**总盈亏**：{总盈亏} 元（{总涨跌%}）
```
